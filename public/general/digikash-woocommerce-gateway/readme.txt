== Description ==
Easily accept DigiKash payments in your WooCommerce store. This plugin adds DigiKash as a payment gateway option at checkout.

== Installation ==
1. Download the plugin ZIP and upload it to your WordPress site (Plugins > Add New > Upload Plugin).
2. Activate the plugin from the Plugins menu.
3. Go to WooCommerce > Settings > Payments and enable "DigiKash".
4. Enter your DigiKash Merchant ID and API Key in the gateway settings.

== Usage ==
- Customers will see DigiKash as a payment option during checkout.
- On order placement, they will be redirected to DigiKash for secure payment.
- After payment, they will return to your site.

== Frequently Asked Questions ==
= Where do I get my DigiKash credentials? =
Login to your DigiKash merchant dashboard to find your Merchant ID and API Key.

= Can I customize the payment button? =
Yes, you can change the title and description from the WooCommerce payment settings.
